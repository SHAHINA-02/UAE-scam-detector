# 🛡️ SHIELD UAE - Scam Detector

AI-powered fraud detection system designed specifically for UAE residents. Protect yourself from SMS, WhatsApp, and email scams using cutting-edge AI and heuristic analysis.

## ✨ Features
- **Cybersecurity Tech Dashboard**: Premium "Command Center" aesthetic.
- **AI-Powered Analysis**: Integrated with Anthropic Claude 3.5 Sonnet.
- **Heuristic Fallback**: Works offline/offline-mode using a custom UAE-specific pattern matching engine.
- **Multi-language Support**: Analysis available in EN, AR, HI, UR, TL, ML.
- **Direct Reporting**: One-click links to Dubai Police (901) and TRA.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript.
- **Backend**: Vercel Serverless Functions (Node.js).
- **AI**: Anthropic API.

## ⚙️ Local Development
1. Clone the repository.
2. Set your API Key: `$env:ANTHROPIC_API_KEY="your_key"`
3. Run `npm install`.
4. Start the server: `node server.js`
